# Support

## Usage Questions and Help

- Open a GitHub Discussion or Issue with clear reproduction steps.
- Include environment details (Rust version, Soroban tooling version, command output).

## Bug Reports

Create a GitHub Issue and include:

- Expected vs actual behavior
- Minimal reproduction steps
- Relevant logs or test output

## Security Vulnerabilities

Please report vulnerabilities privately via GitHub Security Advisories or the repository security contact path. Avoid public disclosure until maintainers coordinate a fix and release.

## Triage and Escalation

- Maintainers triage incoming issues by severity and area.
- Contract safety and release pipeline issues are prioritized for same-day assessment when possible.
- If no response is received within 3 business days, mention maintainers in the issue and link prior context.
